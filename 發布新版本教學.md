# 發布新版本教學 (v2.1.0 更新版)

這是一份給開發者 (您) 閱讀的專屬備忘錄。
當您修改了程式碼，想要發布新版本讓所有使用者能夠「自動接收更新」時，請依照以下步驟操作：

---

## 🚀 最推薦的方法：使用「全自動一鍵發布助手」
我們已經為您整合了所有流程至 `一鍵發布新版本.bat`，這能確保版本號與 GitHub 雲端檔案完全同步：

1. **執行腳本**：雙擊資料夾中的 **`一鍵發布新版本.bat`**。
2. **輸入新版號**：
   - 視窗會顯示目前版本（例如 `2.0.9`）。
   - 直接按 **Enter** 會自動幫您跳號（變為 `2.1.0`），或手動輸入您想要的號碼。
3. **輸入更新內容**：
   - 輸入簡單的文字說明（例如：`優化合併工具流暢度`）。這段文字會出現在 GitHub 的下載頁面上。
4. **全自動處理**：
   - 腳本會自動修改 `main.py` 內部的版本變數。
   - 自動調用 PyInstaller 打包產生新的 `CYT_YTDL.exe`。
   - 自動執行 Git 指令將最新程式碼備份到雲端倉庫。
   - **自動將 `.exe` 上傳到 GitHub Releases**（這是自動更新功能的關鍵）。
5. **最終確認**：
   - 流程結束後，會自動打開瀏覽器顯示剛發布的頁面。
   - **恭喜！** 此時全世界的使用者在開啟舊版程式時，都會收到更新通知。

---

## 🛠️ 手動備援方案 (萬一腳本或網路出錯時)
如果全自動腳本執行失敗，您可以手動進行以下步驟：

1. **修改版本號碼**：
   - 打開 `main.py`，搜尋 `APP_VERSION`，手動修改字串內容（例如改為 `"2.1.1"`）。
2. **手動打包**：
   - 執行 `打包執行檔.bat`。確保產生的檔案位於 `dist/CYT_YTDL.exe`。
3. **手動上傳 GitHub**：
   - 前往 [GitHub Releases 頁面](https://github.com/mathced-com/CYT_YTDL/releases)。
   - 點擊 **Draft a new release**。
   - 設定標籤 (Tag) 為 `v2.1.1`，標題也設為 `v2.1.1`。
   - 將 `dist/CYT_YTDL.exe` 拖入附件區域上傳。
   - 點擊 **Publish release**。

## 📢 附錄：分享軟體時的「安全提示」範本

由於本程式是獨立開發的開源工具，未向微軟購買昂貴的數位簽章，因此他人首次執行時 Windows 會顯示「已保護您的電腦」藍色警告。

您可以**複製以下這段文字**貼給對方，解除他們的疑慮：

> **【關於首次開啟的安全性提示】**
> 
> 哈囉！這支「CYT_YouTube 下載器」是我個人獨立開發的實用小工具。
> 
> 由於這支程式是免費分享，沒有向微軟購買「數位簽章憑證」，因此當您第一次執行 `CYT_YTDL.exe` 時，Windows 會跳出藍色的警告視窗。
> 
> 這只是微軟針對未知發行者的標準防護，**請放心，程式保證絕對無毒！**
> 
> **👉 解除阻擋的方法：**
> 1. 在藍色視窗中，點擊左上角的 **「其他資訊」**。
> 2. 點擊右下角出現的 **「仍要執行」**。
> 
> 這樣就能順利開啟並永久加入白名單囉！祝您使用愉快！
